Veelgestelde vragen

Alles wat je wilt weten.

Kan ik mijn huidige boekingssysteem blijven gebruiken?
Ja. Wij koppelen je website aan je bestaande systeem — Treatwell, Fresha, Salonized of iets anders. Jouw klanten blijven boeken zoals ze gewend zijn, maar via een veel aantrekkelijkere ervaring.
Is de website echt op maat gemaakt?
Ja. Geen standaard uitstraling. Alles wordt afgestemd op jouw zaak, sfeer en doelgroep.
Wat als ik het ontwerp wil aanpassen?
Dat is precies de bedoeling. Wij leveren een eerste ontwerp en passen het aan op jouw wensen — totdat jij écht blij bent.
Werkt de website goed op mobiel?
Ja, we bouwen mobile-first. De meeste klanten zoeken en boeken via hun telefoon.
Kunnen jullie ook hosting regelen?
Ja. We regelen het domein en de hosting voor je — zodat je daar zelf geen omkijken naar hebt.
Wat kost het en hoe snel staat het er?
€999 eenmalig. Geen abonnement, geen verborgen kosten. Zodra alle input compleet is, staat jouw website er snel.
